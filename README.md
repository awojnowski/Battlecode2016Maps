Installation Instructions
-------------------------

1. Open `Terminal`
2. `cd` to the `battlecode-scaffold-master` directory
3. Type `rm -rf maps`
4. Type `git clone git@github.com:awojnowski/Battlecode2016Maps.git`
5. Type `mv Battlecode2016Maps maps`
6. Run Battlecode!
